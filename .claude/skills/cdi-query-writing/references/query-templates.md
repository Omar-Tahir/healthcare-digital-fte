# CDI Query Templates — One Per Opportunity Type

## 1. Acute Kidney Injury (AKI)

```
The patient's creatinine rose from [baseline] to [current]
within [timeframe], which meets KDIGO criteria for acute
kidney injury.

Based on your clinical assessment, does the patient have:
[] Acute kidney injury (AKI) — if so, please specify stage
   and etiology if known
[] Acute on chronic kidney disease
[] Chronic kidney disease progression
[] Creatinine elevation due to other cause: ___________
[] Unable to determine at this time
```

## 2. Sepsis

```
The patient meets SIRS criteria ([specific criteria met])
and has a documented/suspected [infection source].

Based on your clinical assessment:
[] Sepsis is present
[] Severe sepsis with organ dysfunction (specify: _______)
[] Septic shock
[] SIRS without infection / non-infectious etiology
[] Infection present but does not meet sepsis criteria
[] Unable to determine at this time
```

## 3. Heart Failure Type/Acuity

```
Heart failure is documented without specification of type
or acuity. BNP is [value].

Please clarify:
Type: [] Systolic (HFrEF)  [] Diastolic (HFpEF)  [] Combined
Acuity: [] Acute  [] Chronic  [] Acute on chronic
[] Heart failure is not present
[] Unable to determine type/acuity at this time
```

## 4. Respiratory Failure

```
The patient has O2 saturation of [value] and is currently
on [oxygen delivery method]. Documentation notes
[respiratory distress/hypoxia] but does not specify
respiratory failure.

Based on your clinical assessment:
[] Acute respiratory failure (hypoxic / hypercapnic / mixed)
[] Acute on chronic respiratory failure
[] Hypoxia without respiratory failure
[] Other respiratory condition: ___________
[] Unable to determine at this time
```

## 5. Malnutrition

```
The patient has [BMI value / albumin value / weight loss
percentage]. A dietitian [has/has not] been consulted.

Based on your clinical assessment:
[] Severe protein-calorie malnutrition
[] Moderate protein-calorie malnutrition
[] Mild malnutrition
[] Nutritional deficiency not present
[] Unable to determine at this time

If malnutrition is present, please specify etiology if known.
```

```

Based on your clinical assessment:

```

## 8. Diabetes Complications

```

neuropathy / nephropathy / retinopathy / foot ulcer].

documented causal relationship.

[] Yes — diabetic [complication]

```
